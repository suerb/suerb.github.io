正在使用的 Chrome 插件

发表于 2016-06-19

For my lovely girlfriend.

SimpleUndoClose

撤销关闭 tab 的插件，好久没研究过有没有更好的，一直用这个。

UC二维码

UC 自己做的二维码生成工具，功能单一。有替代品，但是没研究过。

Evernote Clips

其实我用的是 Evernote Clearly（印象笔记 - 阅读），但是下架了。这个剪藏更加纯粹。

扇贝插件

这个好像是你介绍我用的？设定快捷键后很不错，但因为不稳定，仅限于速查，很少用，现在逐渐习惯 Apple macOS 的三指来查。

花瓣网页收藏工具

截取、裁剪、标记网页，可以选取部分页面，我最常用的就是整张网页截图，自动滚动页面截取。

Save to Pocket

我一直在用的 Save for later 阅读服务，这是 Chrome 的录入网页插件。

OneTab

可以将 Chrome 的非固定 tabs 一键保存到 OneTab 的列表内，以减少内存占用。

可以对这个 OneTab 保存的列表执行锁定（不能删除）、分享等操作。

Markdown Here

可以在网页中的富文本编辑器内使用 Markdown 语法 ，输入后一键转换。

Drag and Go

可以拖拽网页中的 link 到新 tab 打开。以前的 chrome 没有这功能的……不知现在有没有。

往上拖曳表示后台打开，向下前台打开。左右方向表示放在当前Tab的左边还是右边。

它还包括鼠标手势的，我全部禁用，觉得有点反人类。

Search by Image (by Google)

可以右击网页中的图片来执行 Google image 搜索。灰常方便。

Palettab

我习惯新建一个空白标签页来防止路过的人看到我在看啥网页，也可以顺便让自己放空。这个插件可以在新建标签页中随机显示一组英文字体和颜色。

Imagus

鼠标 hover 于缩略图之上即可原尺寸预览。

豆藤

你介绍给我用的。

暂时就这些了。
