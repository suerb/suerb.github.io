17° 阅读课题组 WOOP

发表于 2016-12-27

Wish

一年内至少完成 20 本书的阅读，且不影响其他专业的书籍进度；

Outcome

  1. 未读过的数，带着问题去读，并解答问题；
  2. 已读过的书，获得新的认知，列举更多实例；
  3. 争取每本书都深入理解，输出高质量读书笔记；最低限度是脑图。
  4. 在书中获得的知识，与工作内容、日常观察的事物相结合，加深理解。

Obstacle

  1. 在看书前，怎样提出问题；
  2. 每天的看书时间不稳定；
  3. 怎样合理安排其他专业的阅读时间；
  4. 怎样高效阅读扫描版书籍，特别是外文书籍；
  5. 读书笔记的写作框架。

Plan

  1. if 拿起书时，无法提出问题，then 去豆瓣、知乎寻找相关书评，或看一下目录，了解书籍的概要；
  2. if 无法知道加班安排，then 挤压午休、刷知乎的时间，或抽取关键章节阅读；
  3. if 专业书籍阅读时间冲突，then 安排优先级，按优先级阅读；
  4. if iPhone 无法满足效率，then 寻找多平台同步的工具并使用 MacBook 阅读；
  5. if 没有好的写作框架，then 至少用问答形式来回答阅读前提出的问题。
